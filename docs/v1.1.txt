Legacy API v1.1

Base URL: http://localhost:4011

Authentication:
- All requests must include header: X-Api-Key

Business objects:
- OrderRecord fields: id, customer_ref, commodity_code_id, amount
- InvoiceRecord fields: id, customer_ref, line_item_code, total

Action: create_order
1) Create order:
   PUT /orders
   Payload JSON:
   {
     "customer_ref": "string",
     "commodity_code_id": "string",
     "amount": number
   }
   Response JSON:
   {
     "id": "string"
   }

2) Read order:
   GET /orders/:id
   Response JSON:
   {
     "id": "string",
     "customer_ref": "string",
     "commodity_code_id": "string",
     "amount": number
   }

Action: create_invoice
1) Create invoice:
   PUT /invoices
   Payload JSON:
   {
     "customer_ref": "string",
     "line_item_code": "string",
     "total": number
   }
   Response JSON:
   {
     "id": "string"
   }

2) Read invoice:
   GET /invoices/:id
   Response JSON:
   {
     "id": "string",
     "customer_ref": "string",
     "line_item_code": "string",
     "total": number
   }

Errors:
- 401 unauthorized
- 404 not_found
- 400 bad_request
